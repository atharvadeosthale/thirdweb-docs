## What are UI Components?

We support various UI components across different platforms for you to use in your apps. These components are built on top of our core SDKs and are designed to be as easy to use as possible. These components are made so you can implement basic Web3 actions, such as connecting wallets and calling contracts, without writing much code and worrying about the underlying details.

## Why use UI Components?

Our UI components handle many complications under the hood instead of putting the hard work on the developer. We also provide customization options, so you have complete flexibility over the UI components and their behaviour in your app.

The best example of this is the Web3 Button. It is a button to perform a certain action on your smart contract, such as buying a drop or minting an NFT. Many complications might appear while implementing this logic from scratch. You need to check if the user's wallet is connected, the contract is loaded correctly, and the user is connected to the correct network. We handle all of this for you. Web3 Button gives you a simple button that checks if everything is in place and then performs the specified action.
